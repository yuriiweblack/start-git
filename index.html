Connect Git to GitHub

Git command:
  1. git status - перевірити поточний стан репозиторію GIT.
  2. git init - ініціалізація папки проекта, виконується один раз.
  3. git add . - файли стають зеленими перед збереженням в commit
  4. git commit -m "назва коміту- інфо про внесені зміни"
  5. git push - відправка збережених змін на GitHub
  6. git log - перевірити наявність збережених комітів
